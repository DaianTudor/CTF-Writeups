# Welcome

### RO:

Flagul are patru părți, iar după identificarea lor, trebuie să aplicăm ROT 13 ca să obținem flagul final:
- **Partea I.** În fișierul `/static/css/main.css` folosit în Homepage.
- **Partea II.** În Homepage, este un text scris alb pe alb, „ascuns”.
- **Partea III.** În scriptul javascript `/static/hidden.js?v=1775752469159`
- **Partea IV.** În tagul cu `id="invisible"`, este o imagine `logo.png` care conține ultima parte.

### EN:

The flag has four parts, and after identifying them, we must apply ROT13 to obtain the final flag:

- **Part I.** In the file `/static/css/main.css` used on the homepage.
- **Part II.** On the homepage, there is some white, “hidden” text.
- **Part III.** In the JavaScript script `/static/hidden.js?v=1775752469159`
- **Part IV.** In the tag with `id="invisible"`, there is an image `logo.png` that contains the final part.
